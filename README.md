# 🌐 Projeto de Rede Doméstica

## Objetivo
Demonstrar configuração e funcionamento de uma rede doméstica.

## Estrutura da rede
- Roteador Wi-Fi
- Dispositivos conectados (celular, computador)

## Configurações realizadas
- DHCP ativado
- Distribuição automática de IP
- Configuração de Wi-Fi

## Problemas resolvidos
- Instabilidade na conexão
- Lentidão na internet

## Tecnologias utilizadas
- Redes TCP/IP
- DHCP
- DNS
